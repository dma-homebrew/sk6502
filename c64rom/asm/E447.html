E447: BASIC vectors
these are copied to RAM from $0300 onwards
E447 .WORD $E38B $0300: error message
E449 .WORD $A483 $0302: BASIC warm start
E44B .WORD $A57C $0304: crunch BASIC tokens
E44D .WORD $A71A $0306: uncrunch BASIC tokens
E44F .WORD $A7E4 $0308: start new BASIC code
E451 .WORD $AE86 $030A: get arithmetic element
